# 병원 온라인 접수 사이트
- 프로젝트명 : Hospital For U
- 근처 병원을 찾아 진료예약을 할 수 있는 웹사이트 입니다.
- 2020-04-20 ~ 2020-05-25
- [바로가기](http://rclass.iptime.org:9999/20AM_HU_MedicalProject_final/)

> ### 주요 기능
* 병원 온라인 접수 및 예약
* 사용자 위치 근처 병원 조회
* 실시간 건강 정보 검색 순위
* 병원 실시간 문의 채팅

>### 부가 기능
* 돌보미 찾기 게시판 CRUD
* 1:1 문의 게시판 CRUD
* 게시판 댓글, 파일 업로드,다운로드
* 회원가입-(단방향 암호화)
* 로그인, ID, 비밀번호 찾기
* 회원-관리자 1:1 채팅
